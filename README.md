# SCUT 习题库

基于课程知识点的习题库设计与实现

## 具体功能

- 用户
  - [] 注册登录
  - [ ] 个人主页
  - [ ] 收藏夹
- 课程
  - [ ] 增删改查
- 知识点
  - [ ] 增删改查
- 题⽬
  - [ ] 增删改查
  - [ ] 筛选题⽬（根据题目难度、题⽬类型等进行筛选）
  - [ ] 收藏题⽬
- 试卷
  - [ ] ⼀键组卷
  - [ ] ⼿动组卷
  - [ ] 下载试卷

## 技术选型

- 框架：[T3 Stack](https://create.t3.gg)
  - [Next.js](https://nextjs.org)
  - [NextAuth](https://next-auth.js.org)
  - [Prisma](https://prisma.io)
  - [Tailwind CSS](https://tailwindcss.com)
  - [tRPC](https://trpc.io)
- 数据库：[MySQL](https://www.mysql.com)
- 部署：[Netlify](https://netlify.com)
- 测试：[Cypress](https://cypress.io)
